Reader – RSS direkt im Browser

Die folgende Oberfläche lädt wesentliche Teile des Python-Skripts Main.py via WebAssembly (Pyodide). Die eigentlichen Feed-Dateien liegen lokal im Repository (z. B. via git pull aktualisiert) und werden direkt aus dem feeds/-Ordner gelesen. Damit entfällt das CORS-Problem, solange Sie die XML-Snapshots regelmäßig per cache_feeds.py erneuern und ins Repository pushen.

Nach dem Klick auf „Feed laden“ wird das Python-Programm gestartet und verarbeitet die lokal vorhandene XML-Datei. Aktualisieren Sie die Dateien per python cache_feeds.py und Git.
